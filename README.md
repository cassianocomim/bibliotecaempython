# Sistema de Controle de Biblioteca

## Sobre o Projeto

Este projeto consiste em um sistema de controle de biblioteca desenvolvido como trabalho final da disciplina de Algoritmos e Programação. Seu objetivo é permitir o gerenciamento de livros por meio de uma interface baseada em menu, possibilitando o cadastro, consulta, alteração e remoção de registros, além do controle de empréstimos e devoluções.

Cada livro possui as seguintes informações:

* Título
* Autor
* Ano de publicação
* Código único de identificação (ID)
* Status (Disponível ou Emprestado)

## Funcionalidades

O sistema oferece as seguintes funcionalidades:

* Cadastro de novos livros
* Consulta de livros por código ou autor
* Alteração dos dados de um livro
* Remoção de livros cadastrados
* Listagem de todos os livros
* Registro de empréstimos
* Registro de devoluções
* Encerramento seguro do programa

## Objetivo Educacional

O projeto foi desenvolvido para aplicar conceitos fundamentais de programação, incluindo:

* Modularização utilizando funções;
* Manipulação de listas e estruturas de dados;
* Validação de entradas;
* Controle de fluxo;
* Organização e manutenção de código;
* Desenvolvimento de sistemas de gerenciamento simples.

## Tecnologias Utilizadas

* Python 3

## Estrutura do Sistema

O sistema foi projetado seguindo boas práticas de programação, evitando o uso de variáveis globais e priorizando a passagem de dados por parâmetros entre funções, tornando o código mais organizado, reutilizável e fácil de manter.
